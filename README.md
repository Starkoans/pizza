
## Stack

- vite
- react-router-dom
- redux-toolkit
- react hook form

## Системные требования

> необходимо скачать [**nodejs/npm**](https://nodejs.org/en/download/)

- **nodejs** > 16
- **npm** > 8

## Как запустить

Сначала нужно установить npm пакеты

```
npm i
```

После чего запустить приложение

```
npm run dev
```
